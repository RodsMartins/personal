# Recruiter Reviewer Checklist

Persona : Recruteur tech / Hiring manager évaluant le profil en 30-60 secondes.

## Première impression (≤ 30 secondes)
- [ ] Le nom, le titre et le positionnement sont clairs dès l'arrivée
- [ ] On comprend immédiatement le domaine d'expertise (Full Stack, Frontend, etc.)
- [ ] Le design inspire le professionnalisme et la rigueur technique
- [ ] Pas de contenu "lorem ipsum" ou placeholder visible

## Parcours professionnel
- [ ] Les expériences sont ordonnées chronologiquement (plus récent en premier)
- [ ] Chaque poste indique : titre, entreprise, lieu, dates
- [ ] Les réalisations sont quantifiées quand possible (%, temps, équipes)
- [ ] La progression de carrière est visible et cohérente
- [ ] Les descriptions utilisent des verbes d'action

## Compétences
- [ ] Les compétences sont organisées par catégorie (Frontend, Backend, DevOps, Outils)
- [ ] Le niveau d'expertise est implicitement clair via le contexte des projets
- [ ] Technologies à jour et pertinentes pour le marché actuel
- [ ] Pas de liste trop longue qui dilue l'impact

## Projets
- [ ] Chaque projet a un titre, une description concise et les technologies utilisées
- [ ] Les projets démontrent une variété de compétences
- [ ] Liens vers le code source et/ou démo fonctionnels
- [ ] Au moins un projet "phare" qui se démarque

## Formation
- [ ] Diplômes et certifications clairement présentés
- [ ] Dates cohérentes avec le parcours professionnel
- [ ] Certifications pertinentes et à jour
- [ ] Type de formation identifiable (diplôme, certification, MOOC)

## Narrative & Différenciation
- [ ] Le "À propos" raconte une histoire, pas juste une liste de compétences
- [ ] Le profil se différencie des autres développeurs
- [ ] La passion et la motivation sont perceptibles
- [ ] Cohérence entre le bio, les expériences et les projets

## Contact & Accessibilité
- [ ] Email ou moyen de contact facilement accessible
- [ ] Liens vers GitHub et LinkedIn fonctionnels
- [ ] Le site se charge rapidement (pas de frustration d'attente)
- [ ] Le site est navigable sur mobile (souvent la première consultation)
