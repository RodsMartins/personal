## Contexte

Création d'un site portfolio personnel pour présenter mon parcours professionnel, mes projets et mes formations.

## Fonctionnalités

- Design sombre et moderne avec animations subtiles
- Architecture statique optimisée pour la performance
- Collections de contenu avec validation Zod
- Prêt pour l'internationalisation (FR/EN)
- Déploiement automatique via GitHub Actions

## Stack technique

Le site utilise **Astro.js** en mode statique avec des CSS custom properties pour le theming, sans aucun framework CSS. Le contenu est géré via les content collections d'Astro avec le glob loader.
