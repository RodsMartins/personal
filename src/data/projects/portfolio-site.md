## Contexte

Conception et développement d'un portfolio professionnel pour présenter mon parcours, mes compétences et mes projets.

## Points clés

- Site statique généré avec Astro 6 pour des performances optimales
- Design system CSS vanilla avec variables custom, sans framework UI
- Déploiement automatisé via GitHub Actions sur GitHub Pages
- Collections de contenu Astro avec schémas Zod pour les projets et formations
- Animations scroll-reveal avec IntersectionObserver, respectant `prefers-reduced-motion`

## Stack technique

Astro pour la génération statique, CSS vanilla avec custom properties pour le theming, TypeScript, déploiement CI/CD via GitHub Actions.
